# Hi, I'm José Luis Castro 👋

**Founder @ [HappyAndroids](https://happyandroids.com)** · **CTO @ [Codefriends](https://www.codefriends.es)** · IT consulting & product development from Málaga, Spain 🇪🇸

For 20+ years I've been building software — from robust Python backends to cloud infrastructure with Docker, AWS and Kubernetes. Currently focused on my own products, AI, and building **Solera Linux**, an immutable and atomic distribution centered on digital sovereignty.

My first piece of software was a pretty ugly little game written in Pascal... and at 18 I ended up working in the Spanish video game industry, on *Blade: The Edge of Darkness*.

> 📬 **Available for freelance projects and consulting** (architecture, backend, AI, infrastructure). Get in touch: [info@happyandroids.com](mailto:info@happyandroids.com)
